# 🍕 ItaPizzaJP — Landing Page

Landing page de uma pizzaria fictícia desenvolvida como projeto de estudo.

## 🔗 Deploy

[Acesse o projeto aqui](https://jpgoguinha-dev.github.io/itapizzahash/)

## 📋 Sobre o projeto

Página institucional da ItaPizzaJP, pizzaria fictícia localizada em Itaberaba, Bahia.
O projeto simula uma landing page real com seções de apresentação, cardápio, promoções e avaliações de clientes.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📄 Seções

- **Home** — apresentação e chamada para ação
- **Sobre nós** — descrição da pizzaria
- **Menu** — listagem de pizzas com preços
- **Promoções** — combos disponíveis
- **Avaliações** — depoimentos de clientes
